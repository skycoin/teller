# event.skycoin.net

https://event.skycoin.net

## Setup

1. Install npm

2. Install Yarn:

```sh
npm install -g yarn
```

3. Install dependencies:

```sh
yarn
```

## Development

Start the development server:

```sh
yarn start
```

Run tests:

```sh
yarn test

# Generate coverage report
yarn test -- --coverage
```

## Build release

Changes should be compiled and committed to the repo.

```sh
yarn build
```
